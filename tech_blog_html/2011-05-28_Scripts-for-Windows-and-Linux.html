I have used a number of scripts to help facilitate a number of tasks: some of them are complex, others simple. Most of these are Windows batch scripts. However, I'm going to add more Linux scripts soon.

There are approximately a hundred scripts in the repository at the time of updating this post. Please visit here: https://github.com/sacsdu/Scripts. and some reusable executable components that are parts of these scripts (helps achieve certain goals). I’m putting a brief summary here.

Windows Scripts

1. getword.cmd: get the pronunciation for specified word and play it with Windows Media Player
2. mac_spoofer: Change MAC and IP to specified host ip
3. install_patches: Install windows updates using patches from specified folder
4. ChangeIntConf: switch between ISPs. In office, I use to easily switch between ISPs using this script. Also it was useful to change my ATA device network settings.
5. ChromeDev_init.cmd & ChromeDev.cmd: set depot tools path and opens a new window
6. Cygwin_init.cmd & Cygwin.cmd: Open a customized Cygwin prompt
7. oglcl.cmd: Script includes all steps necessary to compile opengl files when you have windows sdk and glut installed. It runs the command line version of Visual Studio compiler to compile opengl project.
8. Share folder in network using net share command; delete them: addshares.cmd, delshares.cmd
9. Change IP Address of local area connection: change_ip.cmd
10. Change DNS servers for local area connection: change_dns.cmd
11. Run disk diagnostic tools on all drives of hard-disk and shut the pc down: check & down.cmd
12. Display reminders in message box: displaymsgdone.cmd
13. Incomplete Windows Script for rapidshare link downloads: dnrssingle.cmd Assembly scripts: Longtime ago I was doing some assembly programming. Here are those scripts: MASM_Init.cmd, masm32_cls.cmd, MASM32_Init.cmd, mpl.cmd. If you need more help on these assembly scripts please look have a look here:http://sa-os.blogspot.com/2010/01/assembling-and-linking-assembly.html.

14. Open Jar files: ejar.cmd
15. Generate java bytecodes from java sources and run: Ejava.cmd, Ejavac.cmd
16. Extract single lines from file: file_var.cmd
17. Sink down in disguise: flush_os.cmd
18. Wait for specified timer interval and then notify on top of all windows: mdelay.cmd
19. Open any file with notepad++: npp.cmd
20. Create installer from installer script: nsis.cmd
21. Create new source file and open with visual studio: ocf.cmd
22. Rename all other extension files i.e.; .dat to .mpg:
23. Rename_others_to_mpg.cmd
24. Vista won’t boot because you just reinstalled mbr. So repair it: Repair_vistabootmgr.cmd
25. Start, stop, web services (apache, php, mysql): samp.cmd
26. Want to just ping using the host id: like your ip is 10 so you type sping 10: sping.cmd
27. Stop mssql services consumes too much CPU; start: StopServicesMSSQL.cmd, StartServicesMSSQL.cmd
28. Shutdown PC when specified file is downloaded completely: stdn.cmd
29. Get an svnadmin prompt: svnadmin_init.cmd and svnadmin.cmd
30. Open a stable svn client: SVNDev_init.cmd, SVNDev.cmd
31. Force Windows Media Player to shutdown: wmpkill.cmd
32. This script enables Integrate Internet Download Manager integration after a specified amount of time and then disable on press key: afd.cmd
33. Change MAC Address of Network Interface: change_mac.cmd
34. checkheader.cmd: Get HTTP header for any website. Required installation of netcat.

Linux Scripts

rsdn.sh: provided links on queue.txt The script downlods all links mentioned in queue.txt It supports automated download from rapidshare.

There are some shortcuts in the repository. They points to different applicaiton to assist on commandline launch. For example, if you try command excel.lnk in cmd it will open Microsoft Office Excel application.
