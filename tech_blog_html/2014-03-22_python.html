python.org provides official python installer for different platforms. Our goal in this article is to demonstrate installation of python software in Windows following presenting the way of using Visual Studio as python IDE.

Installation

  1. Download appropriate Windows installer from: https://www.python.org/download/releases/2.7.6 according to your Operating System
    • 64 bit Windows Installer (2.7.6)
    • 32 bit Windows Installer (2.7.6)
  2. Download appropriate Python Tools for Visual Studio (PTVS) installer from: codeplex pytools
    For Visual Studio 13 -     PTVS 2.0 VS 2013.msi
  3. Run Python Windows Installer
  4. Run Python PTVS installer

Setting Up Environment Variable

  1. Go to My Computer and type Control Panel\System and Security\System on location bar as illustrated on the image below:

  2. Click on Advanced system settings as pointed on the image below

  3. Click Environment Variables Button

  4. Click on New Button under User Variables

  5. Add path of python: C:\python27 into environment variable text box after %path%;.

  6. Click OK to apply new settings

Running Python

Using Command Prompt

Fire up a command prompt (by typing cmd on Run Dialog or start menu). Type python you will see python's prompt. Type something there for example,

Using Visual Studio IDE

  1. Fire up Visual Studio
  2. Click on New Project

  4. You can start coding python with Visual Studio. A sample 'hello world' would look like this:

A sum problem

Problem: Given two integers a and b you have to find sum of all odd integers from a through b, inclusively.

The python solution to this problem demonstrates how to take inputs, make comments, run a loop and perform simple arithmetic. 

# Prelude: Thanks S. Atique for enriching the post with a sample code
# Author: Atique (http://tech.saoslab.com/author/Atique)
# Date: June 02, 2013

num1 = input("Enter first number: ")
num2 = input("Enter second number: ")

num2 += 1

sum=0
for x in range (num1,num2):
   if x % 2 == 1:
      sum += x
      print "current sum %d" %sum

print "final sum %d" %sum
